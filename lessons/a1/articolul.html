Structuri simple
Nehotărât = prima dată

Am o casă. / În casă este un pat. / Văd o lampă.

Hotărât = știm exact despre ce vorbim

Casa este mare. / Patul este în dormitor. / Lampa este pe masă.

Negație simplă

Nu am o masă mare. / Scaunul nu este nou. / Baia nu este mică.

Text de lectură — A1

Casa Anei

Ana are o casă mică. Casa este luminoasă. În casă este un dormitor, o bucătărie, un living și o baie.

În dormitor este un pat. Patul este mare. Lângă pat este o lampă. Lampa este albă.

În bucătărie este o masă și sunt niște scaune. Masa este rotundă, iar scaunele sunt noi. În baie este o oglindă. Oglinda este mare.

Conversație

Întrebări simple cu articol hotărât și nehotărât

Nehotărât vs. hotărât

Nehotărât

  • Am o casă.
  • În casă este un pat.
  • Văd o lampă.
  • Cumpăr un scaun.

Hotărât

  • Casa este mare.
  • Patul este în dormitor.
  • Lampa este albă.
  • Scaunul este nou.
Dialog model — acasă
Maria: Ai o casă mare?
Ana: Nu, am o casă mică. Casa este luminoasă.
Maria: Ce ai în dormitor?
Ana: Am un pat și o lampă. Patul este mare, iar lampa este albă.
Maria: Ai o masă în bucătărie?
Ana: Da. Masa este rotundă și scaunele sunt noi.

Practică dialogul cu colegul tău!

Întrebări de conversație — nivel A1
1. Ai o casă sau un apartament?
2. Ce ai în dormitor?
3. Cum este bucătăria?
4. Spune: un pat → patul. Continuă cu trei obiecte.
5. Spune: o masă → masa. Continuă cu trei obiecte.
6. Descrie camera ta folosind 3 forme hotărâte.
Fraze utile pentru conversație
Am un / o...
Am un pat. Am o lampă.
... este în cameră.
Patul este în cameră. Masa este în bucătărie.
... este mare / mic / nou.
Dulapul este mare. Baia este mică.
Forma hotărâtă pentru ... este ...
Forma hotărâtă pentru „un pat” este „patul”.

Dictare

Profesorul citește, elevii scriu forma hotărâtă.

✅

Verificare

Alege 10 cuvinte din casă și adaugă articolul.

Jocuri de rol — practică orală
Rol 1: Ești proprietar. Descrii casa: casa, dormitorul, bucătăria.
Rol 2: Ești vizitator. Întrebi: „Ce este în cameră?”
Rol 3: Ești la magazin. Ceri o masă, un scaun și o lampă.
Rol 4: Profesorul spune forma nehotărâtă, elevul spune forma hotărâtă.

✍️ Temă de scriere — 30–50 de cuvinte

Scrie un text scurt despre casa ta. Folosește cel puțin 8 forme hotărâte: casa, camera, patul, masa, scaunul, lampa, baia, oglinda.

Model: Casa mea este... / Dormitorul are... / Bucătăria este...

Mini provocări
Spune 5 cuvinte cu „un” și transformă-le cu „-ul”.
Spune 5 cuvinte cu „o” și transformă-le cu „-a”.
Fă 3 propoziții cu articol nehotărât.
Fă 3 propoziții cu articol hotărât.
