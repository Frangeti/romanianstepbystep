Articolul hotărât și nehotărât: Casa mea

Romanian Step by Step · Nivel A1 · Fișă de lucru PDF

1. Teorie pe scurt

Articol nehotărât: un / o. Îl folosim când vorbim prima dată despre ceva: Am o casă. În casă este un pat.

Articol hotărât: se pune la finalul cuvântului: casa, patul, masa, scaunul. Îl folosim când știm exact despre ce vorbim.

Exemplu: Am o masă. Masa este în bucătărie.

4. Text scurt

Casa mea

Am o casă mică. Casa este luminoasă. În casă este un dormitor, o bucătărie, un living și o baie.

În dormitor este un pat. Patul este mare. Lângă pat este o lampă. Lampa este albă. În bucătărie este o masă. Masa este rotundă. În baie este o oglindă. Oglinda este mare.

4.a. Întrebări

5.d. Adevărat sau fals

  1. „o casă” devine „casa”.
  2. „un pat” devine „pata”.
  3. „o masă” devine „masa”.
  4. „un scaun” devine „scaunul”.
